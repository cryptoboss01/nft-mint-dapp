### - Start

`cd server`

`yarn start`

### - Stacks

Fastify, TypeScript, Nft.storage, 

## Smart Contract

### - Compile

`npx hardhat compile`

### - Units Test

`npx hardhat test`

### - Deploy

`npx hardhat run scripts/deploy.ts --network ropsten`

### - Stack

Solidity, Openzeppelin, Hardhat, Typechain, ERC721
